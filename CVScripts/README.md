# In This Folder

1. Website - Next.js app that is the frontend
2. Vision playground - Python scripts where I try to see what I can do as far as object tracking goes with traditional methods
3. Vision prod - Actual production versions for object tracking

# Stuff You'll Need

1. `pyenv` to match my Python version
2. `pip` for requirements
3. `rust` and `cargo` via `rustup`
4. Docker

Fortunately, I have created `install_deps` scripts for these things for Mac and Linux, so you can just run those and you'll be set. Have Windows? Too bad.
